# Messages Module

## What is the Messages Module?

The Messages Module is your communication hub within Diligince.ai. It enables real-time messaging with vendors, team members, and other stakeholders related to your procurement activities.

## Key Features

- **Direct Messaging**: One-on-one conversations
- **Group Chats**: Multi-party discussions
- **Context Linking**: Messages linked to requirements, POs
- **File Sharing**: Send documents in chat
- **Notifications**: Real-time alerts
- **Message History**: Full conversation archive

## How to Send a Message

### Sending a Direct Message
1. Click "New Message" button
2. Search for recipient:
   - Team member
   - Vendor contact
3. Type your message
4. Click Send (or press Enter)

### Sending in an Existing Conversation
1. Click on the conversation
2. Type in the message input
3. Press Enter or click Send

## How to Use Contextual Messages

### Requirement Discussion
When you need to discuss a specific requirement:
1. Open the requirement details
2. Click "Comments" or "Discussion"
3. Messages are linked to that requirement
4. All stakeholders can participate

### PO Discussion
For purchase order related queries:
1. Open PO details
2. Use the chat/discussion feature
3. Messages are linked to that PO
4. Vendor sees only what's shared with them

## Common Questions

**Q: Can vendors see all my messages?**
A: No, vendors only see messages in conversations where they're participants. Internal team messages are private.

**Q: How long are messages stored?**
A: Messages are stored permanently. You can access full history anytime.

**Q: Can I unsend a message?**
A: You can delete a message, but if the recipient has already read it, they've seen it. Delete removes it from the conversation view.

**Q: How do I know if a message was read?**
A: Read receipts may be shown (depends on settings). Blue checkmarks typically indicate read status.

**Q: Can I message a vendor before they submit a quotation?**
A: Only if you have their contact. Public requirements don't reveal your identity until you choose to engage.

**Q: Are message attachments secure?**
A: Yes, all attachments are encrypted and stored securely. Only conversation participants can access them.

**Q: Can I export chat history?**
A: Depending on your role and settings, you may be able to export conversation history for record-keeping.
