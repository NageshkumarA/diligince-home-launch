# Analytics Module

## What is the Analytics Module?

The Analytics Module provides comprehensive reports, insights, and data visualization for your procurement activities. It helps you make data-driven decisions by analyzing trends, performance, and spending patterns.

## Key Features

- **Interactive Dashboards**: Visual data representations
- **Custom Reports**: Generate tailored reports
- **Trend Analysis**: Track patterns over time
- **Performance Metrics**: KPIs and scorecards
- **Export Capabilities**: Download reports in various formats
- **Scheduled Reports**: Automated report delivery

3. Navigate to specific sections

## How to Use Filters

### Date Range Filter
1. Click the date picker
2. Select predefined range:
   - This Week
   - This Month
   - This Quarter
   - This Year
   - Custom
3. Dashboard updates

### Category Filter
1. Select categories to include
2. Multiple selection allowed
3. Filter applies to all charts

### Vendor Filter
1. Search for specific vendors
2. Select one or more
3. View vendor-specific data

### Status Filter
1. Choose statuses to include
2. Applies to relevant charts

## How to Generate Custom Reports

### Using Report Builder
1. Go to Analytics → Report Builder
2. Select report type:
   - Requirements Report
   - Quotations Report
   - PO Report
   - Vendor Report
   - Spend Report
3. Choose fields to include
4. Apply filters
5. Preview report
6. Generate

### Report Customization
- Select columns to display
- Choose grouping (by vendor, category, date)
- Set sorting order
- Apply date filters
- Add calculations (totals, averages)

## How to Export Reports

### Export Formats
- **PDF**: Formatted document with charts
- **Excel (XLSX)**: Spreadsheet with data
- **CSV**: Raw data for further analysis

### Exporting
1. View the report or dashboard
2. Click Export button
3. Select format
4. File downloads

### Scheduling Reports
1. Go to Report Settings
2. Click "Schedule New Report"
3. Configure:
   - Report type
   - Frequency (Daily, Weekly, Monthly)
   - Recipients (email addresses)
   - Format
4. Save schedule
5. Reports auto-delivered

## Key Performance Indicators (KPIs)

### Procurement KPIs
| KPI | Description | Target |
|-----|-------------|--------|
| Avg. Quotes per Req | Number of quotes received | >3 |
| Quote-to-PO Ratio | % of quotes converted to PO | >20% |
| Cycle Time | Req creation to PO issue | <14 days |
| On-time Delivery | % delivered on schedule | >95% |
| Budget Variance | Actual vs budget | <10% |

### How to Track KPIs
1. View KPI dashboard
2. Green = On target
3. Yellow = Needs attention
4. Red = Below target
5. Click for detailed breakdown

## Common Questions

**Q: How often is analytics data updated?**
A: Data is updated in near real-time. Most charts refresh when you load the page.

**Q: Can I create custom dashboards?**
A: Depending on your plan and role, custom dashboard creation may be available.

**Q: Who can access analytics?**
A: Access is role-based. Admins see all; others may see limited based on permissions.

**Q: How far back does historical data go?**
A: All data from your organization's start is available. Use date filters to access historical periods.

**Q: Can I share reports with others?**
A: Yes, export as PDF/Excel and share, or use scheduled reports to email automatically.

**Q: Are benchmarks available?**
A: Industry benchmarks may be available in premium plans, showing how you compare to similar organizations.

**Q: Can I drill down into charts?**
A: Yes, click on chart elements to see detailed data. For example, click a category in pie chart to see transactions.
