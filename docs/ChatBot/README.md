## Platform Summary

**Diligince.ai** is a B2B procurement platform that connects:
- **Industry Users** (Buyers) - Companies that need services/products
- **Vendors** - Service providers and suppliers
- **Professionals** - Individual experts and consultants

The platform streamlines the entire procurement lifecycle:
1. Create requirements → 2. Receive quotations → 3. Compare and approve → 4. Issue purchase orders → 5. Track delivery → 6. Complete payment
