# User Account Settings

## What is User Account Settings?

User Account Settings is where you manage your personal profile, notification preferences, and security settings. Unlike the company Settings module, these settings affect only your individual account.

## Key Features

- **Profile Management**: Your personal information
- **Notification Preferences**: How you receive alerts
- **Security Settings**: Password and 2FA
- **Session Management**: Active logins

## Notifications Tab

### Notification Channels
Configure where you receive notifications:

| Channel | Description |
|---------|-------------|
| **In-App** | Notifications within Diligince.ai |
| **Email** | Notifications to your email |
| **SMS** | Text messages to your phone |

### Notification Types
Choose which events trigger notifications:

**Procurement Updates**
- New quotation received
- Quote approved/rejected
- PO status change
- Delivery update

**Approvals**
- Pending approval request
- Approval completed
- Rejection notification

**Messages**
- New message received
- Mentions (@you)
- Group chat updates

**System**
- Security alerts
- Account changes
- Platform announcements

### How to Configure Notifications
1. Go to Account Settings â†’ Notifications
2. For each category:
   - Toggle on/off
   - Select channels (Email, SMS, In-App)
3. Set frequency:
   - Real-time
   - Daily digest
   - Weekly summary
4. Configure quiet hours (optional):
   - Set start/end time
   - No notifications during quiet hours
5. Save preferences

### Two-Factor Authentication (2FA)

#### What is 2FA?
Extra security layer requiring a code in addition to password.

#### How to Enable 2FA
1. Go to Account Settings â†’ Security
2. Find "Two-Factor Authentication"
3. Toggle to Enable
4. Choose method:
   - SMS (code to phone)
   - Email (code to email)
5. Verify with a test code
6. 2FA is now active

#### How 2FA Works
1. Enter your password
2. Receive 6-digit code
3. Enter code within 5 minutes
4. Access granted

#### How to Disable 2FA
1. Go to Security settings
2. Toggle 2FA off
3. Confirm with current 2FA code
4. 2FA disabled

#### Recovery Codes
When you enable 2FA:
1. System generates recovery codes
2. Save these codes securely
3. Use if you can't receive 2FA code
4. Each code works once

### Active Sessions

#### What are Active Sessions?
List of devices/browsers where you're logged in.

#### Viewing Sessions
1. Go to Account Settings â†’ Security
2. See "Active Sessions" section
3. Each session shows:
   - Device type
   - Browser
   - Location (approximate)
   - Last active time
   - Current session marked

#### Revoking a Session
1. Find the session to remove
2. Click "Revoke" or "Sign Out"
3. That session is immediately logged out
4. Use if you see suspicious activity

#### Sign Out All Devices
1. Click "Sign Out All Other Sessions"
2. Confirm
3. All sessions except current are logged out
4. Use if account compromised

## Common Questions

**Q: Can I change my email address?**
A: Email is your unique identifier and cannot be changed. Create a new account if you need a different email.

**Q: Why can't I edit some profile fields?**
A: Some fields are managed by your organization's admin. Contact them for changes.

**Q: How do I reset my password if I forgot it?**
A: Click "Forgot Password" on the login page. Reset link is sent to your email.

**Q: What if I lose my phone and can't receive 2FA codes?**
A: Use a recovery code. If you don't have them, contact your admin to reset 2FA.

**Q: Why am I getting too many notifications?**
A: Adjust preferences in Notifications tab. Switch to digest mode or disable non-essential alerts.

**Q: How do I know if someone else accessed my account?**
A: Check Active Sessions. Revoke any sessions you don't recognize. Enable 2FA for added security.

**Q: Can I download my data?**
A: Data export requests should be made to your admin or through platform support.
